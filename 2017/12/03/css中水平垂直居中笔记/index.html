css水平垂直居中笔记

2017.12.03 weily  热度 ℃

众所周知，如果不给一个元素添加固定高度，那么它就会自动适应其内容的高度。
怎么让宽度也具有类似行为？
example: 想要达到的一个效果为图片和外部盒子（box）居中于fatherBox,并且box宽度自适应于图片。

tip：平常写代码时会遇到这样一个问题，如果div里嵌套一个img元素且div的高度是由img的高度来撑开的，那么div的高度总会比img的高度多几个px.
解决办法：给 img 加一个 display:block;
